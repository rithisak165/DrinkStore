# Drink Store POS System

## 🚀 Overview
The Drink Store POS System is a fullstack web application designed to manage a real-world drink store business. It provides functionality for product management, order processing, and payment handling.

This system is designed to reflect a real-world production environment, featuring scalable architecture, role-based access control, and business logic commonly used in retail POS systems.

## 🔥 Features

### 👤 Role-Based Access Control
- Admin: Full system control (products, users, orders, settings)
- Customer: Browse products, place orders, and make payments

---

### 🛍 Product Management
- Add, edit, and delete products
- Manage product details and availability

---

### 👥 User Management
- Admin can update user email and reset passwords
- Secure authentication system

---

### 🧾 Order & Payment System
- Customers can place orders and complete payments
- Real-time notification when payment is completed
- Supports ABA / bank transfer payment

---

### 📊 Dashboard & Analytics
- View total revenue and sales performance
- Filter data by daily, weekly, monthly, and yearly

---

### ⏰ Store Control System
- Set store working hours (e.g., 8:00 AM – 8:00 PM)
- Automatically disable ordering when store is closed

---

### 🔔 Notifications
- Admin receives updates when orders are placed or paid

---

## 🛠 Tech Stack

- Frontend: React.js  
- Backend: Laravel (REST API)  
- Database: MySQL  
- Styling: Tailwind CSS
